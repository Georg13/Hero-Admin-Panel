Hero Admin Panel — панель управления героями с CRUD-операциями и фильтрацией

Мини-приложение для управления списком героев с возможностью создания, просмотра, удаления и фильтрации. Данные сохраняются в JSON-файле через JSON Server.

### 🦸‍♂️ Управление героями
- ✅ **Создание** — добавление новых героев через форму
- ✅ **Просмотр** — список всех героев в виде карточек
- ✅ **Удаление** — удаление героев с подтверждением
- ✅ **Сохранение** — автоматическое сохранение в JSON файл

### 🔍 Фильтрация
- 🔥 По стихиям (огонь, вода, земля, воздух)
- ⚡ Мгновенное обновление списка
